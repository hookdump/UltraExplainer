Why the gate runs first
Classify + embed cost ≈ $0.004 per item. Gating to the relevant 9% turns a $400 run into $36 — the single biggest lever in the pipeline.
The relevance check is one cheap boolean call (~120ms). The heavy workers never queue behind irrelevant traffic, so p95 end-to-end stays under 2s.
Fewer junk items means tighter clusters: precision on the top-20 clusters rose from 0.71 to 0.88 once noise stopped diluting centroids.
Multi-tenant. Each agent gets an isolated database at {agentDir}/intelligence/feedback.db — the collapse point above — with per-agent config overlay and credential isolation.
